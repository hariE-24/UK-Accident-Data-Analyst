# UK Accident Data Analyst Dashboard

## Project Overview
This project analyzes UK road accident data using Power BI to identify:
- Accident trends
- Casualty severity
- Vehicle types
- Weather conditions
- Road surface impact
- Yearly accident patterns

## Tools Used
- Power BI
- Excel / CSV
- DAX
- Data Cleaning
- Data Visualization

## Dashboard Features
- KPI Cards
- Year-wise analysis
- Casualty severity analysis
- Vehicle type analysis
- Interactive filters and slicers
- Trend analysis

## Key Insights
- Most accidents occur in urban areas
- Car vehicles contribute highest accident count
- Wet road conditions increase severity
- Peak accidents happen during evening hours
